# RFC index

This is a catalog of every RFC in `main` — all `IMPLEMENTED`, `REJECTED`, and `SUPERSEDED` proposals. The **RFC number** is assigned here, after merge; it appears nowhere else, and no RFC directory is ever renamed. Numbers run in a single sequence across all categories and outcomes.

An RFC earns its number only once it is decided and merged. While an RFC is still open — `#proposed` or `#accepted` — it lives at `rfc/<category>/<slug>/` with no number, and does not appear here.

The **decision date** is the date the RFC was decided — its approval date for implemented RFCs, its rejection date for rejected ones — as recorded in the RFC document.
